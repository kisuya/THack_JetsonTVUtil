# THack_JetsonTVUtil
TV Watching guide Application 


## Features
* Face Detection and calculate distance from tv
* Pose Estimation and detect wrong pose 
  * use openpose
* send event to set-top box via http
  
  
## System Architecture
* OS - Ubuntu 16.04.02
* Qt with OpenCV Based application
  * Qt - 5.5.1
  * OpenCV - 2.4.11
